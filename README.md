# 🌍 Agentic AI Travel Planner

An intelligent travel planning application built using **LangGraph**, **LangChain**, and **LLMs** that generates personalized travel itineraries based on a user's destination and interests.

---

## 📌 Project Overview

Planning a trip often requires searching through multiple websites, blogs, and guides to create a suitable itinerary. This project simplifies the process by using an AI-powered agent that understands user preferences and automatically generates a customized travel plan.

The Travel Planner takes user inputs such as:

* Destination city
* Travel interests (food, museums, adventure, shopping, nature, etc.)

and produces a detailed itinerary tailored to those preferences.

---

## ✨ Features

* 🏙️ Personalized itinerary generation
* 🤖 Agent-based workflow using LangGraph
* 💬 Natural language interaction using LLMs
* 🧠 Conversation memory using LangChain messages
* 🔄 Stateful execution through LangGraph
* 🎯 Interest-based recommendations
* 🌐 Streamlit-ready frontend integration
* 📓 Initially developed and tested in Jupyter Notebook
